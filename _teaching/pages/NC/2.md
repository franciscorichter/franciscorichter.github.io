This lecture focuses on floating‑point number systems, roundoff error, catastrophic cancellation, and numerical stability, with mathematical rigor and runnable code demonstrations.

## Core Concepts
- Floating‑point (IEEE 754), machine epsilon, precision limits
- Roundoff error accumulation, standard model, Kahan summation
- Catastrophic cancellation and stable reformulations
- Numerical stability: forward vs backward error, design principles

## Key Results
```
ε_mach = 2^(-52) ≈ 2.22×10^(-16) (double precision)
|fl(s_n) - s_n| ≤ γ_n Σ|x_i|,  where γ_n = nε_mach/(1 - nε_mach)
Forward Error ≤ κ(f) × Backward Error
```
