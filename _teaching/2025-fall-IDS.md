This course provides an introduction to the key concepts, techniques, and tools of data science, with a focus on practical applications in artificial intelligence and computational science. Topics typically include:

- Data collection and cleaning
- Exploratory data analysis
- Statistical inference and modeling
- Machine learning fundamentals
- Data visualization
- Applications in real-world scenarios

For more information, visit the [official USI course page](https://search.usi.ch/it/corsi/35275499/introduction-to-data-science-msc).
