This course provides an introduction to the key concepts, techniques, and tools of data science, with a focus on practical applications in artificial intelligence and computational science. Topics typically include:

- Data collection and cleaning
- Exploratory data analysis
- Statistical inference and modeling
- Machine learning fundamentals
- Data visualization
- Applications in real-world scenarios

## Course Outline (expanded)

- **Foundations and tooling:** Python, Jupyter, environments, version control, reproducibility.
- **Data access:** CSV/JSON/Parquet, SQL, web scraping, REST APIs; data provenance and documentation.
- **Wrangling and cleaning:** tidy data principles, missing data strategies, outlier detection, data validation.
- **EDA and visualization:** univariate/bivariate analysis, feature relationships, best practices, storytelling.
- **Statistical basics:** sampling, bias/variance, estimation, confidence intervals, hypothesis testing.
- **Supervised learning:** regression (linear, regularization), classification (logistic, trees, ensembles), calibration.
- **Model evaluation:** metrics, cross‑validation, resampling, learning curves, overfitting/underfitting.
- **Pipelines and features:** preprocessing, encoders, transformers, feature selection, model persistence.
- **Unsupervised learning:** clustering (k‑means, hierarchical), dimensionality reduction (PCA), anomaly detection.
- **Responsible AI (intro):** interpretability basics, fairness notions, data ethics and privacy considerations.
- **Communication:** dashboards, reports, and effective visual narratives for technical and non‑technical audiences.
